## 项目概述

拾光记是一款功能丰富的多平台习惯追踪和个人管理应用，帮助用户养成良好习惯，提高工作效率，记录生活点滴。

### 技术栈

- **框架**：Flutter 3.0+
- **语言**：Dart 3.0+
- **状态管理**：BLoC + Riverpod
- **数据库**：SQLite + Supabase
- **支持平台**：Android、iOS、Windows、macOS、Linux、Web

## 功能特性

### 核心功能
- 待办事项管理
- 习惯打卡与追踪
- 富文本日记
- 番茄钟专注计时
- 数据统计与分析
- 积分兑换系统
- 回收站功能

### 高级功能
- Supabase 云端同步
- 多设备实时同步
- 电商系统（星星商店）
- 物流管理
- 搜索增强
- 数据导入导出
