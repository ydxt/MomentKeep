# 数据导入导出功能文档

## 功能概述

为拾光记 (Moment Keep) 应用添加了完整的数据导入导出功能，支持待办事项、习惯、日记等核心业务数据的备份和恢复。

## 实现的功能

### 1. 数据导出

- **导出格式**：ZIP 压缩包，包含结构化 JSON 文件
- **支持模块**：
  - ✅ 分类数据 (categories)
  - ✅ 待办事项 (todos) - 包含子任务
  - ✅ 习惯追踪 (habits) - 包含打卡记录
  - ✅ 日记 (journals) - 自动解密内容
  - ✅ 番茄钟 (pomodoros)
  - ✅ 学习计划 (plans)
  - ✅ 成就 (achievements)
  - ✅ 回收站 (recycle_bin)
  - ✅ 媒体文件 (media) - 图片、音频、视频

### 2. 数据导入

- **导入流程**：
  1. 选择 ZIP 导出文件
  2. 自动解析并验证文件格式
  3. 预览文件信息和数据统计
  4. 确认导入
  5. 按依赖顺序导入数据（分类 → 其他数据）
  6. 显示导入结果报告

- **安全保障**：
  - 版本兼容性检查
  - 文件格式验证
  - 导入前预览
  - 详细的导入结果报告

## 使用方式

### 访问入口

1. 打开应用
2. 进入"设置"页面
3. 在"其他"分组中点击"数据导入导出"

### 导出数据

1. 点击"开始导出"按钮
2. 在导出选项页面中选择要导出的模块（支持全选/取消全选）
3. 点击"开始导出"按钮
4. 等待导出完成
5. 通过分享对话框保存或发送导出文件

### 导入数据

1. 点击"开始导入"按钮
2. 选择 ZIP 格式的导出文件
3. 查看文件信息和数据统计
4. 点击"开始导入"按钮
5. 等待导入完成并查看结果报告

## 技术实现

### 核心服务 (ImportExportService)

- **单例模式**：确保全局唯一实例
- **数据源整合**：
  - SharedPreferences：待办事项、习惯、分类、回收站
  - SQLite 数据库：日记、番茄钟、计划、成就
  - 文件系统：媒体文件（图片、音频、视频）
- **加密处理**：日记内容自动解密/加密
- **ZIP 打包**：使用 archive 库进行 ZIP 文件创建和解析

### 数据结构

导出文件结构：
```
momentkeep_export_[timestamp].zip
├── manifest.json          # 导出元信息
├── categories.json        # 分类数据
├── todos.json             # 待办事项
├── habits.json            # 习惯及打卡
├── journals.json          # 日记（已解密）
├── pomodoros.json         # 番茄钟
├── plans.json             # 计划
├── achievements.json      # 成就
├── recycle_bin.json       # 回收站
└── media/                 # 媒体文件
    ├── images/
    ├── audio/
    └── video/
```

为以下实体添加了 `toJson()` 和 `fromJson()` 方法：

- `HabitReminder`：习惯提醒

## 数据完整性保障

### 导入顺序

1. **分类** (categories) - 优先导入，因为其他数据可能引用
2. **待办事项** (todos)
3. **习惯** (habits)
4. **日记** (journals) - 包含媒体文件路径处理
5. **番茄钟** (pomodoros)
6. **计划** (plans)
7. **成就** (achievements)
8. **回收站** (recycle_bin)

### 媒体文件处理

- 导出时：复制媒体文件到 ZIP 的 media/ 目录
- 导入时：解压到临时目录，更新日记内容中的路径

### 错误处理

- 文件不存在检查
- 格式验证
- 单条数据导入失败不影响其他数据
- 详细的错误信息反馈

## 已知限制和改进方向

### 当前限制

1. **Web 平台不支持**：文件系统访问限制
2. **媒体文件路径**：导入后需要手动更新日记中的媒体路径
3. **数据覆盖**：导入会追加数据，不会清空现有数据

### 未来改进

1. ⭐ 添加数据合并/覆盖选项

6. ⭐ 支持选择性导入（只导入部分模块）

## 故障排除

### 常见问题

**Q: 导出失败？**
A: 检查存储空间，确保有足够的空间用于导出文件

**Q: 导入失败？**
A: 确保导出文件格式正确，未损坏，版本兼容

**Q: 媒体文件丢失？**
A: 导出时确保勾选"媒体文件"选项

**Q: 数据导入后不完整？**
A: 查看导入结果报告，确认所有模块都已导入
