# 数据存放策略 - 总结报告

## 1. 项目概述

### 1.1 项目背景
- **项目名称**: MomentKeep
- **任务目标**: 制定数据存放策略，明确服务器（cloud目录）和本地存储的数据类型

- **拆分了可执行的子任务**

## 2. 存储架构设计

### 2.1 目录结构

#### 2.1.1 服务器目录（cloud目录）
```
用户设置目录/
└── cloud/
    ├── database/           # 服务器数据库
    │   ├── moment_keep_users.db   # 用户主数据库
    │   └── moment_keep_ecommerce.db  # 电商数据库
    ├── files/              # 服务器文件存储
    │   ├── products/       # 商品图片/视频
    │   ├── users/          # 用户头像/认证文件
    │   └── system/         # 系统文件
    └── logs/               # 服务器日志
```

#### 2.1.2 本地目录
```
用户设置目录/
└── MomentKeep/
    ├── default/            # 默认数据
    │   ├── database/       # 默认数据库
    │   └── store/          # 商店默认数据
    ├── [用户ID]/           # 每个用户独立目录
    │   ├── images/         # 个人图片
    │   ├── audio/          # 个人音频
    │   ├── video/          # 个人视频
    │   └── files/           # 其他个人文件
    └── local_database/     # 本地数据库
        ├── moment_keep.db  # 本地应用数据库
        └── cache.db         # 缓存数据库
```

## 3. 数据分类策略

### 3.1 必须放在服务器的数据

#### 3.1.1 用户相关数据
- **用户主表数据**：用户ID、用户类型、状态、登录记录等
- **买家扩展数据**：昵称、头像、联系方式、会员等级、积分等
- **商家扩展数据**：商家名称、营业执照、联系人信息、店铺信息等
- **管理员扩展数据**：管理员信息、权限等
- **信用积分数据**：买家信用积分、卖家信用积分
- **认证文件**：商家营业执照、身份证等认证材料

#### 3.1.2 电商相关数据
- **商品数据**：商品信息、SKU、规格、库存等
- **订单数据**：订单信息、订单商品、订单状态等
- **交易数据**：支付记录、退款记录、物流信息等
- **优惠券数据**：优惠券信息、用户优惠券等
- **红包数据**：红包活动、领取记录等
- **购物卡数据**：购物卡信息、交易记录等
- **物流数据**：物流公司、物流跟踪等

#### 3.1.3 系统相关数据
- **系统配置**：全局设置、版本信息等
- **日志数据**：操作日志、错误日志等
- **统计数据**：销售统计、用户统计等

### 3.2 应该放在本地的数据

#### 3.2.1 个人隐私数据
- **日记数据**：智能日记内容、心情、标签等
- **习惯数据**：习惯设置、打卡记录等
- **待办事项**：待办列表、完成状态等
- **个人设置**：应用设置、主题偏好等
- **个人文件**：个人照片、录音、视频等

#### 3.2.2 缓存数据
- **商品缓存**：浏览过的商品信息
- **图片缓存**：商品图片、用户头像等
- **搜索历史**：搜索关键词、浏览历史等
- **临时文件**：编辑中的文件、上传中的文件等

#### 3.2.3 离线数据
- **离线商品**：下载的商品信息
- **离线订单**：历史订单缓存
- **离线日记**：本地日记备份

## 4. 数据同步策略

### 4.1 实时同步
- **用户认证数据**：登录、注册、修改密码等
- **订单数据**：创建订单、支付状态等
- **商品数据**：商品信息、库存更新等
- **信用积分**：积分变动、等级变更等

### 4.2 定期同步
- **用户扩展数据**：个人资料更新
- **日记数据**：定期备份到云端
- **习惯数据**：定期同步打卡记录
- **统计数据**：销售数据、用户行为数据等

### 4.3 手动同步
- **大文件**：视频、高清图片等
- **批量数据**：批量商品导入、批量订单处理等
- **备份数据**：手动备份本地数据到云端
